# Como Ultilizar.

1. Clone o repositor1.0io ou baixe o codigo fonte.
2. Abra o Terminal ou prompt de Comando e navegue até a pasta raiz.
3. Utilize o camando abaixo para restaurar as dependências do projeto.

```
dotnet restore
```


4. Utilize o comando abaixo para compilar o projeto.
```
dotnet build --configuration Release
```

5. Utilize o comando abaixo para executar o projeto.
```
dotnet run -- project ListaDeExercicios
```

6. Utilize o comando abaixo para executar os testes.
